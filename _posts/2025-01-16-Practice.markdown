 `_posts` 디렉토리에서 해당 글을 작성 할 수 있음
`jekyll serve --watch` 명령어로 빌드 가능

새로운 글을 작성하려면 `_posts` 디렉토리에 `YYYY-MM-DD-name-of-post.ext` 와 같은 방식으로 마크다운 작성

코드 사용 방법

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}

링크 작성 방법
{% highlight ruby %}
# [원하는 텍스트][링크]
{% endhighlight %}
